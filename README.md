# MFIR
A reinforcement learning framework utilizing LSTM and a dual-branch neural network to simulate and defend against cross-layer cyber-physical attacks in the partially observable scenario, based on the Tennessee Eastman (TE) process model.
## Prerequisites
To run this simulation, you need the following installed:
* Python 3.8+
* PyTorch
* Gymnasium
* Stable-Baselines3
* MATLAB (with Simulink and the MATLAB Engine API for Python installed)
## Note
Ensure your MATLAB paths inside cross_layer_sim2.py are correctly pointing to the model and sim directories on your local machine before running.
